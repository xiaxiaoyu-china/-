## 神途游戏客户端GUI类接口-骨骼动画控件学习总结

### 一、控件概述

骨骼动画是由互相连接的“骨头”组成的骨架结构，通过控制、改变骨头的位置、朝向和放大和缩小来为角色生成的动画。相比较于传统的帧动画，骨骼动画的资源是一些小的角色部件（如头、手、胳膊、腰等），具有更好的灵活性和复用性。

**适用平台**：手游（2D/3D）、端游（2D/3D）、H5

---

### 三、实际应用示例

#### 示例1：创建和播放骨骼动画

```lua
-- 初始化函数
function OnInit()
    -- 创建主窗口
    local main_wnd = GUI:WndCreateWnd(0, "MainWindow", 100, 100, 400, 300)

    -- 创建骨骼动画控件
    local ske_anim = GUI:SkeAnimCreate(main_wnd, "SkeAnim", 150, 100, 100, 200)

    -- 设置动画播放
    GUI:SkeAnimSetAnimation(ske_anim, 0, "idle", true)

    -- 设置动画属性
    GUI:SkeAnimSetTimeScale(ske_anim, 1.2)
    GUI:SkeAnimSetDebugBonesEnabled(ske_anim, true)

    -- 显示窗口
    GUI:WndSetVisible(main_wnd, true)

    -- 设置定时器切换动画
    GUI:WndAddTimer(main_wnd, 1001, 3000, -1, "OnSwitchAnimation")
end

-- 定时器回调函数
function OnSwitchAnimation(wnd, timer_id)
    local ske_anim = GUI:WndFindChild(wnd, "SkeAnim")

    -- 获取当前播放的动画名称
    -- 注意：这里需要根据实际接口获取当前动画名称，以下为伪代码
    local current_anim = "idle"

    if current_anim == "idle" then
        GUI:SkeAnimSetAnimation(ske_anim, 0, "walk", true)
    else
        GUI:SkeAnimSetAnimation(ske_anim, 0, "idle", true)
    end
end

-- 登出函数
function OnQuitGame()
    -- 清理资源
    GUI:WndClose("MainWindow")
end
```

#### 示例2：骨骼动画事件回调

```lua
-- 初始化函数
function OnInit()
    -- 创建主窗口
    local main_wnd = GUI:WndCreateWnd(0, "MainWindow", 100, 100, 400, 300)

    -- 创建骨骼动画控件
    local ske_anim = GUI:SkeAnimCreate(main_wnd, "SkeAnim", 150, 100, 100, 200)

    -- 设置动画播放
    GUI:SkeAnimSetAnimation(ske_anim, 0, "attack", false)

    -- 注册事件回调
    GUI:SkeAnimSetEventCallBack(ske_anim, "OnAnimationEvent")

    -- 显示窗口
    GUI:WndSetVisible(main_wnd, true)
end

-- 骨骼动画事件回调
function OnAnimationEvent(event_type, param1, param2)
    -- 事件类型判断
    if event_type == "animation_complete" then
        -- 动画播放完成，切换到待机动画
        local main_wnd = GUI:WndFindChild(nil, "MainWindow")
        local ske_anim = GUI:WndFindChild(main_wnd, "SkeAnim")
        GUI:SkeAnimSetAnimation(ske_anim, 0, "idle", true)

        GUI:SysMsg_SendPromptMsg("攻击动画播放完成")
    elseif event_type == "animation_event" then
        -- 动画事件触发
        GUI:SysMsg_SendPromptMsg(string.format("动画事件：%s, 参数：%s, %s",
            event_type, param1, param2))
    end
end

-- 登出函数
function OnQuitGame()
    -- 清理资源
    GUI:WndClose("MainWindow")
end
```

#### 示例3：骨骼动画皮肤切换

-- 初始化函数
function OnInit()
    -- 创建主窗口
    local main_wnd = GUI:WndCreateWnd(0, "MainWindow", 100, 100, 400, 300)

    -- 创建骨骼动画控件
    local ske_anim = GUI:SkeAnimCreate(main_wnd, "SkeAnim", 150, 100, 100, 200)

    -- 设置动画播放
    GUI:SkeAnimSetAnimation(ske_anim, 0, "idle", true)

    -- 创建切换皮肤的按钮
    local btn_warrior = GUI:ButtonCreate(main_wnd, "WarriorBtn", 50, 250, 80, 25)
    GUI:ButtonSetTextM(btn_warrior, "战士皮肤")
    GUI:WndRegistScript(btn_warrior, "OnLButtonUp", "OnWarriorSkinClick")

    GUI:ButtonSetTextM(btn_mage, "法师皮肤")

    GUI:ButtonSetTextM(btn_archer, "弓箭手皮肤")
    GUI:WndRegistScript(btn_archer, "OnLButtonUp", "OnArcherSkinClick")

    -- 显示窗口
    GUI:WndSetVisible(main_wnd, true)

function OnWarriorSkinClick(wnd, handle)

### 四、学习建议与最佳实践

#### 1. 学习路径

1. **基础阶段**：学习骨骼动画控件的创建和基本播放方法
2. **进阶阶段**：学习动画切换、皮肤设置和事件回调
3. **高级阶段**：学习骨骼动画与游戏角色系统的集成

#### 2. 最佳实践

- 合理使用骨骼动画的循环播放和事件回调，实现复杂的动画逻辑
- 对不同职业和角色类型使用不同的动画皮肤，提高游戏视觉效果
- 定期清理骨骼动画资源，避免内存泄漏
- 对骨骼动画的性能进行合理优化，确保游戏运行流畅

#### 3. 性能优化

- 避免在短时间内频繁切换动画
- 对不需要显示骨骼和插槽的场景，关闭调试显示
- 合理使用动画皮肤，避免加载过多的皮肤资源

通过以上学习总结，您可以全面掌握神途游戏客户端中GUI类接口-骨骼动画控件的使用方法，开发出功能完善、视觉效果良好的游戏角色动画系统。
