## 神途游戏客户端系统接口学习总结

### 一、概述

系统接口是神途游戏客户端中用于处理系统级功能和工具类操作的核心接口集合。这些接口提供了文本处理、剪贴板操作、系统信息获取等基础功能，是开发游戏系统功能的基础。

**适用平台**：手游（2D/3D）、端游（2D/3D）、H5

---

### 二、接口分类与使用方法

#### 1. 文本处理接口

| 接口名 | 中文名 | 返回值类型 | 功能描述 | 示例代码 |
|--------|--------|----------|----------|----------|
| CutString | 分割字符串 | bool | 按指定的字体和宽度，将字符串分割为两段 | `if CL:CutString("system", "12345", 20) then dbg(LuaRet[1]..","..LuaRet[2]) end` |
| GetRichEditFontStrWidth | 获取文本的像素宽度 | int | 获取一段文本在指定字体下的像素宽度 | `local width = CL:GetRichEditFontStrWidth("system", "测试文本")` |
| GetTimeStrWithFormat | 转换时间戳为指定格式文本 | string | 将时间戳转换为指定格式的文本 | `local time_str = CL:GetTimeStrWithFormat(tostring(os.time()), "%Y-%m-%d")` |

---

#### 2. 剪贴板操作接口

| 接口名 | 中文名 | 返回值类型 | 功能描述 | 示例代码 |
|--------|--------|----------|----------|----------|
| CopyTextToClipBoard | 复制文本到剪贴板 | 无 | 将指定文本复制到系统剪贴板 | `CL:CopyTextToClipBoard("需要复制的文本")` |
| GetTextToClipBoard | 获取剪切板中的文本 | string | 获取系统剪贴板中的文本内容 | `local text = CL:GetTextToClipBoard()` |
| SetPasteboardText | 设置剪切板文本内容 | 无 | 设置系统剪贴板的文本内容 | `CL:SetPasteboardText("设置剪贴板内容")` |

---

### 三、实际应用示例

#### 示例1：文本处理与格式化

```lua
-- 初始化函数
function OnInit()
    -- 测试字符串分割功能
    if CL:CutString("system", "这是一个测试字符串", 50) then
        dbg(string.format("分割后的字符串：第一段='%s', 第二段='%s'", LuaRet[1], LuaRet[2]))
    else
        dbg("字符串分割失败")
    end

    -- 测试文本宽度计算
    local text_width = CL:GetRichEditFontStrWidth("system", "测试文本宽度计算")
    dbg(string.format("文本宽度：%dpx", text_width))

    -- 测试时间格式化
    local current_time = os.time()
    local date_str = CL:GetTimeStrWithFormat(tostring(current_time), "%Y-%m-%d %H:%M:%S")
    dbg(string.format("当前时间：%s", date_str))

    -- 设置定时器定期更新时间
    CL:AddTimer(1001, 1000, -1, "OnUpdateTime")
end

-- 定时器回调函数
function OnUpdateTime(timer_id)
    local current_time = os.time()
    local time_str = CL:GetTimeStrWithFormat(tostring(current_time), "%H:%M:%S")
    dbg(string.format("当前时间：%s", time_str))
end

-- 登出函数
function OnQuitGame()
    -- 停止定时器
    CL:RemoveTimer(1001)
end
```

#### 示例2：剪贴板操作

```lua
-- 初始化函数
function OnInit()
    -- 测试复制到剪贴板
    CL:CopyTextToClipBoard("测试剪贴板功能")

    -- 测试获取剪贴板内容
    local clipboard_text = CL:GetTextToClipBoard()
    dbg(string.format("剪贴板内容：%s", clipboard_text))

    -- 测试设置剪贴板内容
    CL:SetPasteboardText("新的剪贴板内容")
    dbg(string.format("修改后的剪贴板内容：%s", CL:GetTextToClipBoard()))

    -- 创建测试按钮
    local main_wnd = GUI:WndCreateWnd(0, "TestWindow", 100, 100, 300, 200)
    GUI:WndSetVisible(main_wnd, true)

    local btn_copy = GUI:ButtonCreate(main_wnd, "CopyBtn", 50, 50, 100, 30)
    GUI:ButtonSetTextM(btn_copy, "复制文本")
    GUI:WndRegistScript(btn_copy, "OnLButtonUp", "OnCopyText")

    local btn_paste = GUI:ButtonCreate(main_wnd, "PasteBtn", 50, 100, 100, 30)
    GUI:ButtonSetTextM(btn_paste, "粘贴文本")
    GUI:WndRegistScript(btn_paste, "OnLButtonUp", "OnPasteText")
end

-- 复制文本按钮点击事件
function OnCopyText(wnd, handle)
    local test_text = "从按钮复制的文本"
    CL:CopyTextToClipBoard(test_text)
    CL:SysMsg_SendPromptMsg(string.format("已复制文本：%s", test_text))
end

-- 粘贴文本按钮点击事件
function OnPasteText(wnd, handle)
    local clipboard_text = CL:GetTextToClipBoard()
    CL:SysMsg_SendPromptMsg(string.format("剪贴板内容：%s", clipboard_text))
end

-- 登出函数
function OnQuitGame()
    GUI:WndClose("TestWindow")
end
```

### 三、学习建议与最佳实践

#### 1. 学习路径

1. **基础阶段**：学习文本处理接口，了解字符串分割和宽度计算方法
2. **进阶阶段**：学习剪贴板操作接口，掌握文本复制和粘贴功能
3. **高级阶段**：学习如何组合多个系统接口，实现复杂的系统功能

#### 2. 最佳实践

- 在使用文本处理接口时，注意字体设置和字符编码问题
- 对剪贴板操作进行适当的错误处理，避免操作失败导致的问题
- 合理使用定时器，避免频繁更新时间导致性能问题

#### 3. 常见使用场景

- 游戏内聊天系统的文本格式化
- 角色属性面板的信息展示
- 系统设置界面的文本显示
- 游戏日志记录和时间戳格式化

通过以上学习总结，您可以全面掌握神途游戏客户端中系统接口的使用方法，开发出功能完善、用户体验良好的游戏系统功能。
