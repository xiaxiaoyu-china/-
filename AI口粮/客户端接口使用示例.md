## 神途游戏客户端接口使用示例

### 一、概述

本文件包含了神途游戏客户端接口的详细使用示例，覆盖了场景操作、角色管理、UI开发、通信与事件处理等功能。这些示例代码可以帮助开发者快速理解和使用客户端接口，提高开发效率。

**适用平台**：根据接口类型不同，支持手游（2D/3D）、端游（2D/3D）或H5

### 三、通信与事件处理机制

```lua
-- 事件监听与处理
-- 在GameEventDefine.lua中定义事件
LUA_EVENT_ONPLAYLEVELUP = 1  -- 玩家升级事件

-- 在脚本中添加事件监听
function OnGameEvent(event_id)
    if event_id == LUA_EVENT_ONPLAYLEVELUP then
        -- 从LuaParam中获取参数
        local new_level = LuaParam[1]
        dbg("玩家升级到" .. new_level .. "级")

        -- 处理升级逻辑
        CL:SysMsg_SendWarnMsg("恭喜你升级到" .. new_level .. "级!")
    elseif event_id == LUA_EVENT_ONAPPLYITEM then
        -- 物品使用事件
        local item_name = LuaParam[1]
        local item_guid = LuaParam[2]
        local item_id = LuaParam[3]
        dbg("使用了物品: " .. item_name .. ", GUID: " .. item_guid)
    end
end

-- 系统通知
CL:SysMsg_SendPromptMsg("这是一个提示消息")
CL:SysMsg_SendWarnMsg("这是一个警告消息")
CL:SysMsg_SendCenterMsg("这是一个中心消息")
```
