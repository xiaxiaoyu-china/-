## 神途客户端接口学习总结

### 一、核心API概览

神途客户端接口是游戏开发中的核心功能模块，提供了丰富的API接口和通信机制，支持多平台兼容（2D手游/3D手游/2D端游/3D端游/H5）。通过这些接口，开发者可以实现场景操作、角色管理、UI开发、通信与事件处理等功能。

**接口分类**：
- 核心API：基础接口和通用功能
- 场景操作：地图场景相关操作
- 角色操作：玩家角色相关操作
- 全局函数：全局回调函数和事件处理
- 系统接口：系统级和工具类接口
- UI操作：界面控件相关操作
- 通信与事件：逻辑事件和通信机制

**平台兼容性说明**：
| 平台 | 2D手游 | 3D手游 | 2D端游 | 3D端游 | H5 |
|------|--------|--------|--------|--------|----|
| 支持程度 | 全面支持 | 全面支持 | 全面支持 | 全面支持 | 部分支持 |
| 注意事项 | 需使用GUI:前缀接口 | 需使用GUI:前缀接口 | 直接使用CL:前缀接口 | 直接使用CL:前缀接口 | 部分接口限制 |

**地图场景操作**：
```lua

local scene_x = LuaRet[1]
local scene_y = LuaRet[2]
```

#### 3. 全局函数与事件处理（全局函数.lua）

**场景事件回调**：
```lua
-- 场景遮挡回调（仅3D支持）
function BehindWall(_Value)
    if _Value == 1 then
        -- 场景遮挡触发
        dbg("角色进入遮挡区域")
    else
        -- 场景遮挡结束
        dbg("角色离开遮挡区域")
    end
end

-- 选中目标判断（LogicFunction.lua）
function CheckNpcClicked(_GUID, _Type, _ID, _PosX, _PosY, _ClickX, _ClickY)
    dbg(string.format("选中目标: %s, 类型: %d, 位置: (%d,%d)",
        _GUID, _Type, _PosX, _PosY))

    if _Type == 2 then  -- 怪物
        if _PosX == _ClickX and _PosY == _ClickY then
            return true  -- 允许交互
        end
        return false  -- 不允许交互
    end

    return true  -- 默认允许交互
end

-- 自定义系统消息（LogicFunction.lua）
function CreateSysAnnItem(parent, msg, param)
    -- 创建自定义系统消息
    local item = CL:CreateSysAnnItem(parent)
    CL:SetSysAnnItemText(item, msg)
    CL:SetSysAnnItemAnimation(item, 1)  -- 动画类型1
    return item
end
```

### 三、通信与事件处理机制

#### 1. 逻辑事件系统

**事件监听与处理**：
```lua
-- 在GameEventDefine.lua中定义事件
LUA_EVENT_ONPLAYLEVELUP = 1  -- 玩家升级事件

-- 在脚本中添加事件监听
function OnGameEvent(event_id)
    if event_id == LUA_EVENT_ONPLAYLEVELUP then
        -- 从LuaParam中获取参数
        local new_level = LuaParam[1]
        dbg("玩家升级到" .. new_level .. "级")

        -- 处理升级逻辑
        CL:SysMsg_SendWarnMsg("恭喜你升级到" .. new_level .. "级!")
    elseif event_id == LUA_EVENT_ONAPPLYITEM then
        -- 物品使用事件
        local item_name = LuaParam[1]
        local item_guid = LuaParam[2]
        local item_id = LuaParam[3]
        dbg("使用了物品: " .. item_name .. ", GUID: " .. item_guid)
    end
end

-- 系统通知
CL:SysMsg_SendPromptMsg("这是一个提示消息")
CL:SysMsg_SendWarnMsg("这是一个警告消息")
CL:SysMsg_SendCenterMsg("这是一个中心消息")
```
