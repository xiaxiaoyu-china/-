# 神途游戏客户端接口学习总结 - 窗口控件接口

## 概述

窗口控件是神途游戏客户端中最基础且最重要的UI组件之一，它是其他所有UI控件的容器和基础。窗口控件提供了界面的框架结构，用于组织和布局各种交互元素，如按钮、编辑框、图片、列表等。

### 核心功能

- **界面容器**：窗口是其他UI控件的载体，负责组织和管理界面布局
- **交互框架**：提供用户与游戏界面进行交互的基础架构
- **视觉层级**：通过窗口的Z轴坐标和层级管理，实现界面元素的遮挡关系
- **响应式设计**：支持窗口的移动、大小调整和显示/隐藏操作
- **事件处理**：提供窗口级别的事件响应和处理机制

### 应用场景

- **主界面窗口**：游戏主菜单、角色信息面板等
- **功能窗口**：背包、技能、任务等功能界面
- **弹窗提示**：系统提示、对话框、确认框等
- **自定义界面**：根据游戏需求创建的特殊功能界面

## 基础操作

### 1. 窗口创建

#### 接口定义
```lua
-- 创建普通窗口
GUI:WndCreateWnd(_ParentHandle, _WndName, _ResID, _X, _Y)
-- 或使用封装接口
WndCreateWnd(_ParentHandle, _Info, _WndName, _ResID, _X, _Y)
```

#### 参数说明
- `_ParentHandle`：父窗口句柄，通常为LuaGlobal["AttachPartent"]（主界面）
- `_WndName`：窗口名称，用于标识窗口
- `_ResID`：窗口资源ID，用于加载预定义的UI资源
- `_X`：窗口相对于父窗口的X坐标
- `_Y`：窗口相对于父窗口的Y坐标

#### 示例代码
```lua
local _Parent = LuaGlobal["AttachPartent"] -- 获取主界面句柄
local _GUIHandle = GUI:WndCreateWnd(_Parent, "MyCustomWindow", 1900000003, 100, 100)

if _GUIHandle ~= 0 then
    dbg("窗口创建成功，句柄为：" .. _GUIHandle)
else
    dbg("窗口创建失败")
end
```

### 2. 窗口属性设置

#### 位置与大小
```lua
-- 设置窗口位置
WndSetPosM(_Handle, _Info, _X, _Y)

-- 设置窗口大小
WndSetSizeM(_Handle, _Info, _Width, _Height)

-- 同时设置位置和大小
WndSetPosSizeM(_Handle, _Info, _X, _Y, _Width, _Height)
```

#### 显示与隐藏
```lua
-- 设置窗口可见性
WndSetVisibleM(_Handle, _Info, _Visible) -- true为可见，false为不可见

-- 获取窗口可见性
local _IsVisible = WndGetVisibleM(_Handle, _Info)
```

#### 启用与禁用
```lua
-- 设置窗口可用性
WndSetEnableM(_Handle, _Info, _Enable) -- true为可用，false为禁用

-- 获取窗口可用性状态
local _IsEnabled = WndGetEnableM(_Handle, _Info)
```

#### 窗口标题
```lua
-- 设置窗口文本
WndSetTextM(_Handle, _Info, _Text)

-- 获取窗口文本
local _Text = WndGetTextM(_Handle, _Info)
```

#### 背景与透明度
```lua
-- 设置窗口背景图片
WndSetTexture(_Handle, _Info, _TextureID)

-- 设置窗口透明度
WndSetOpacity(_Handle, _Info, _Opacity) -- 范围0-255，0为完全透明
```

### 3. 窗口控制

#### 关闭窗口
```lua
-- 关闭并销毁窗口
GUI:WndClose(_Handle)

-- 或使用封装接口
WndClose(_Handle, _Info)
```

#### 激活与焦点
```lua
-- 设置窗口激活状态
WndSetActiveM(_Handle, _Info, _Active)

-- 设置窗口获得焦点
WndSetFocusM(_Handle, _Info)
```

#### 模态窗口
```lua

```

## 事件处理

### 1. 事件类型

窗口支持多种事件类型，常用的包括：

| 事件ID | 事件名称 | 描述 |
|--------|----------|------|
| RDWndBaseCL_mouse_lbClick | 左键点击 | 鼠标左键点击窗口时触发 |
| RDWndBaseCL_mouse_lbDown | 左键按下 | 鼠标左键按下时触发 |
| RDWndBaseCL_mouse_lbUp | 左键释放 | 鼠标左键释放时触发 |
| RDWndBaseCL_mouse_rbClick | 右键点击 | 鼠标右键点击窗口时触发 |
| RDWndBaseCL_mouse_move | 鼠标移动 | 鼠标在窗口上移动时触发 |
| RDWndBaseCL_wndActive | 窗口激活 | 窗口获得焦点时触发 |
| RDWndBaseCL_wndDeactive | 窗口失活 | 窗口失去焦点时触发 |
| RDWndBaseCL_wndClose | 窗口关闭 | 窗口关闭时触发 |

### 2. 事件注册与取消

#### 事件注册
```lua
-- 注册窗口事件回调函数
WndRegistScript(_Handle, _Info, _MsgID, _FuncName)
```

#### 事件取消
```lua
-- 取消注册窗口事件回调函数
WndUnRegistScript(_Handle, _Info, _MsgID, _FuncName)
```

#### 示例代码
```lua
local _Parent = LuaGlobal["AttachPartent"]
local _GUIHandle = GUI:WndCreateWnd(_Parent, "MyWindow", 1900000003, 100, 100)

if _GUIHandle ~= 0 then
    -- 注册左键点击事件
    WndRegistScript(_GUIHandle, nil, RDWndBaseCL_mouse_lbClick, "OnWindowClick")

    -- 注册窗口关闭事件
    WndRegistScript(_GUIHandle, nil, RDWndBaseCL_wndClose, "OnWindowClose")
end

-- 窗口点击事件处理函数
function OnWindowClick(_Handle)
    dbg("窗口被点击，句柄：" .. _Handle)
    WndSetTextM(_Handle, "", "窗口已被点击")
end

-- 窗口关闭事件处理函数
function OnWindowClose(_Handle)
    dbg("窗口正在关闭，句柄：" .. _Handle)
end
```

local function CreateSettingsPanel()
    local _Parent = LuaGlobal["AttachPartent"]

    -- 创建主窗口
    local _SettingsWnd = GUI:WndCreateWnd(_Parent, "SettingsPanel", 1900000003, 200, 200)

    if _SettingsWnd ~= 0 then
        -- 设置窗口大小

        WndSetTextM(_SettingsWnd, "", "游戏设置")

        if _CloseBtn ~= 0 then
            WndSetTextM(_CloseBtn, "", "关闭")
            WndRegistScript(_CloseBtn, nil, RDWndBaseCL_mouse_lbClick, "CloseSettingsPanel")

### 3. 创建一个对话框窗口

```lua
local function CreateMessageBox(title, message, callback)
    local _Parent = LuaGlobal["AttachPartent"]

    -- 创建对话框窗口
    local _MsgBox = GUI:WndCreateWnd(_Parent, "MessageBox", 1900000003, 300, 200)

    if _MsgBox ~= 0 then
        -- 设置窗口大小
        WndSetSizeM(_MsgBox, "", 300, 180)

        -- 设置窗口标题
        WndSetTextM(_MsgBox, "", title)

        -- 创建消息文本控件
        local _MsgText = GUI:ButtonCreate(_MsgBox, "MsgText", 1900000047, 50, 50)
        if _MsgText ~= 0 then
            WndSetTextM(_MsgText, "", message)
            WndSetEnableM(_MsgText, "", false)
        end

        -- 创建确认按钮
        local _ConfirmBtn = GUI:ButtonCreate(_MsgBox, "ConfirmBtn", 1850000307, 80, 120)
        if _ConfirmBtn ~= 0 then
            WndSetTextM(_ConfirmBtn, "", "确认")
            WndRegistScript(_ConfirmBtn, nil, RDWndBaseCL_mouse_lbClick, "OnConfirmClick")
            -- 保存回调函数
            WndSetParam(_MsgBox, "", callback or "DefaultMessageBoxCallback")
        end

        -- 创建取消按钮
        local _CancelBtn = GUI:ButtonCreate(_MsgBox, "CancelBtn", 1850000307, 180, 120)
        if _CancelBtn ~= 0 then
            WndSetTextM(_CancelBtn, "", "取消")
            WndRegistScript(_CancelBtn, nil, RDWndBaseCL_mouse_lbClick, "OnCancelClick")
        end

        return _MsgBox
    else
        return 0
    end
end

-- 确认按钮点击事件
function OnConfirmClick(_Handle)
    local _Parent = WndGetParent(_Handle, "")
    local _Callback = WndGetParam(_Parent, "")

    -- 调用回调函数
    if _Callback and type(_G[_Callback]) == "function" then
        _G[_Callback](true)
    end

    -- 关闭窗口
    GUI:WndClose(_Parent)
end

-- 取消按钮点击事件
function OnCancelClick(_Handle)
    local _Parent = WndGetParent(_Handle, "")
    local _Callback = WndGetParam(_Parent, "")

    -- 调用回调函数
    if _Callback and type(_G[_Callback]) == "function" then
        _G[_Callback](false)
    end

    -- 关闭窗口
    GUI:WndClose(_Parent)
end

-- 默认回调函数
function DefaultMessageBoxCallback(confirmed)
    if confirmed then
        dbg("用户点击了确认")
    else
        dbg("用户点击了取消")
    end
end

-- 使用示例
local function ShowTestMessage()
    CreateMessageBox("提示", "这是一个测试对话框", "TestMessageBoxCallback")
end

function TestMessageBoxCallback(confirmed)
    if confirmed then
        dbg("测试对话框：用户确认操作")
    else
        dbg("测试对话框：用户取消操作")
    end
end

-- 调用显示函数
ShowTestMessage()
```

## 优化与性能考虑

### 1. 窗口创建与销毁

#### 优化策略
```lua
-- 避免频繁创建和销毁窗口
-- 可以使用隐藏/显示替代创建/销毁
local _CachedWindow = nil

function GetCachedWindow()
    if not _CachedWindow or _CachedWindow == 0 then
        _CachedWindow = CreateMyWindow()
    end

    return _CachedWindow
end

function ShowWindow()
    local _Wnd = GetCachedWindow()
    if _Wnd ~= 0 then
        WndSetVisibleM(_Wnd, "", true)
    end
end

function HideWindow()
    local _Wnd = GetCachedWindow()
    if _Wnd ~= 0 then
        WndSetVisibleM(_Wnd, "", false)
    end
end
```

### 2. 事件处理优化

#### 事件节流
```lua
-- 避免在高频事件（如鼠标移动）中执行复杂操作
local _LastMoveTime = 0

function OnWindowMouseMove(_Handle, _X, _Y)
    local _CurrentTime = os.clock()

    -- 限制事件处理频率（每50ms处理一次）
    if _CurrentTime - _LastMoveTime > 0.05 then
        -- 执行实际的移动处理逻辑
        HandleWindowMove(_Handle, _X, _Y)
        _LastMoveTime = _CurrentTime
    end
end
```

### 3. 资源管理

#### 图片资源释放
```lua
-- 在窗口关闭时释放占用的资源
function OnWindowClose(_Handle)
    -- 释放背景图片资源
    WndSetTexture(_Handle, "", 0)

    -- 移除所有子控件
    local _ChildCount = WndGetChildCount(_Handle, "")
    for i = 0, _ChildCount - 1 do
        local _ChildHandle = WndGetChildByIndex(_Handle, "", i)
        if _ChildHandle ~= 0 then
            GUI:WndClose(_ChildHandle)
        end
    end

    -- 取消事件注册
    WndUnRegistScript(_Handle, nil, RDWndBaseCL_mouse_lbClick, "OnWindowClick")
end
```

## 常见问题与解决方案

### 1. 窗口无法显示

#### 问题原因
- 父窗口句柄无效
- 窗口可见性设置为false
- 窗口大小为0或位置超出屏幕
- 窗口层级设置不当

#### 解决方案
```lua
local function DebugWindowVisibility(_Handle)
    if _Handle == 0 then
        dbg("窗口句柄无效")
        return false
    end

    local _Parent = WndGetParent(_Handle, "")
    if _Parent == 0 then
        dbg("无父窗口或父窗口已被销毁")
    end

    local _Visible = WndGetVisibleM(_Handle, "")
    if not _Visible then
        dbg("窗口可见性为false")
    end

    local _Enable = WndGetEnableM(_Handle, "")
    if not _Enable then
        dbg("窗口可用性为false")
    end

    local _X, _Y, _Width, _Height = WndGetPosSizeM(_Handle, "")
    if _Width == 0 or _Height == 0 then
        dbg("窗口大小为0")
    end

    return true
end
```

### 2. 窗口无法响应点击事件

#### 问题原因
- 窗口不可用（Enable为false）
- 事件未正确注册
- 窗口被其他控件遮挡
- 窗口未获得焦点

#### 解决方案
```lua
local function CheckWindowEvents(_Handle)
    -- 检查窗口可用性
    local _Enable = WndGetEnableM(_Handle, "")
    if not _Enable then
        dbg("窗口不可用")
        WndSetEnableM(_Handle, "", true)
    end

    -- 检查是否有事件注册
    -- 可以通过遍历事件ID检查
    local _HasClickEvent = false

    -- 尝试重新注册事件
    if not _HasClickEvent then
        WndRegistScript(_Handle, nil, RDWndBaseCL_mouse_lbClick, "OnWindowClick")
        dbg("重新注册点击事件")
    end
end
```

### 3. 窗口拖动卡顿

#### 问题原因
- 拖动时执行了复杂操作
- 窗口透明度过高
- 重绘频率过高
- 窗口内容复杂

#### 解决方案
```lua
-- 优化拖动性能
function OnWindowStartDrag(_Handle)
    -- 隐藏复杂内容
    local _ComplexCtrl = WndGetChildByName(_Handle, "", "ComplexControl")
    if _ComplexCtrl ~= 0 then
        WndSetVisibleM(_ComplexCtrl, "", false)
    end
end

function OnWindowEndDrag(_Handle)
    -- 显示复杂内容
    local _ComplexCtrl = WndGetChildByName(_Handle, "", "ComplexControl")
    if _ComplexCtrl ~= 0 then
        WndSetVisibleM(_ComplexCtrl, "", true)
    end
end
```
