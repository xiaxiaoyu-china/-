# 服务端接口使用示例

本文档包含神途游戏服务端所有接口的使用示例，按功能分类整理，方便开发者快速查找和使用。

## 系统接口使用示例

### 系统管理

#### 获取玩家账号名
```lua
local account_name = lualib:AccountName(player)
lualib:Warn("玩家账号名: " .. account_name)
```

#### 添加定时器
```lua
-- 添加定时器，5秒后触发一次
lualib:AddTimer(player, 101, 5000, 1, "on_timer")

-- 定时器回调函数
function on_timer(obj, timer_id)
    lualib:Warn("定时器触发")
end
```

#### 打印调试日志
```lua
lualib:Debug("调试信息: " .. variable)
lualib:Warn("警告信息: " .. variable)
lualib:Error("错误信息: " .. variable)
lualib:Print("普通信息: " .. variable)
```

#### 获取在线玩家列表
```lua
local online_players = lualib:GetAllOnlinePlayerNames()
lualib:Warn("在线玩家数量: " .. #online_players)
for i, name in ipairs(online_players) do
    lualib:Warn("玩家" .. i .. ": " .. name)
end
```

#### 字符串处理
```lua
-- 字符串分割
local str = "a,b,c,d"
local parts = string.split(str, ",")
for i, part in ipairs(parts) do
    lualib:Warn("分割部分" .. i .. ": " .. part)
end

-- 字符串替换
local new_str = string.replace(str, ",", ";")
lualib:Warn("替换后字符串: " .. new_str)

-- 字符串截取
local sub_str = string.left(str, 3)
lualib:Warn("左截取: " .. sub_str)
```

#### 文件操作
```lua
-- 读取文件内容
local file_content = lualib:IO_GetFileText("data.txt")
lualib:Warn("文件内容: " .. file_content)

-- 写入文件
lualib:IO_SetFileText("data.txt", "新内容")

-- 追加内容
lualib:IO_AppendFile("data.txt", "\n追加内容")

-- 检查文件是否存在
if lualib:IO_FileExists("data.txt") then
    lualib:Warn("文件存在")
else
    lualib:Warn("文件不存在")
end
```

#### 时间操作
```lua
-- 获取当前时间
local current_time = lualib:Now()
local time_str = lualib:Time2Str(current_time)
lualib:Warn("当前时间: " .. time_str)

-- 获取服务器运行时间
local run_time = lualib:GetSvrRunTime()
lualib:Warn("服务器运行时间: " .. run_time .. "秒")
```

## 对象通用接口使用示例

### 对象属性操作

#### 获取和设置对象变量
```lua
-- 设置对象变量
lualib:SetInt(player, "level", 10)
lualib:SetStr(player, "name", "测试角色")

-- 获取对象变量
local level = lualib:GetInt(player, "level")
local name = lualib:GetStr(player, "name")
lualib:Warn("角色等级: " .. level .. ", 角色名称: " .. name)

-- 删除对象变量
lualib:DeleteInt(player, "level")
lualib:DeleteStr(player, "name")
```

#### 获取对象位置
```lua
local x = lualib:X(player)
local y = lualib:Y(player)
local map_guid = lualib:MapGuid(player)
lualib:Warn("玩家位置: (" .. x .. ", " .. y .. "), 地图GUID: " .. map_guid)
```

#### 获取对象名称和类型
```lua
local name = lualib:Name(player)
local obj_type = lualib:GUIDType(player)
lualib:Warn("对象名称: " .. name .. ", 类型: " .. obj_type)
```

## 玩家接口使用示例

### 角色属性操作

#### 获取和设置角色属性
```lua
-- 获取角色属性
local level = lualib:Level(player)
local job = lualib:Job(player)
local hp = lualib:Hp(player)
local mp = lualib:Mp(player)
lualib:Warn("等级: " .. level .. ", 职业: " .. job .. ", HP: " .. hp .. ", MP: " .. mp)

-- 设置角色等级
lualib:SetLevel(player, 100)

-- 设置角色生命值
lualib:SetHp(player, 1000)
```

#### 获取和设置角色装备
```lua
-- 获取角色装备
local weapon = lualib:Weapon(player)
local armor = lualib:Armor(player)
local helmet = lualib:Helmet(player)
lualib:Warn("武器: " .. weapon .. ", 衣服: " .. armor .. ", 头盔: " .. helmet)

-- 让玩家穿装备
lualib:ApplyEquipEx(player, item_guid)

-- 让玩家脱装备
lualib:DisableEquip(player, equip_site)
```

#### 获取和设置角色技能
```lua
-- 获取角色技能
local skills = lualib:GetAllSkills(player)
lualib:Warn("角色技能数量: " .. #skills)
for i, skill_id in ipairs(skills) do
    lualib:Warn("技能" .. i .. ": " .. skill_id)
end

-- 给角色添加技能
if lualib:AddSkill(player, "巨石火轮") then
    lualib:SysMsg_SendWarnMsg(player, "技能学习成功")
else
    lualib:SysMsg_SendWarnMsg(player, "技能学习失败")
end

-- 删除角色技能
lualib:DelSkill(player, "巨石火轮")
```

#### 获取和设置角色BUFF
```lua
-- 获取角色BUFF列表
local buff_list = lualib:GetBuffList(player)
lualib:Warn("角色BUFF数量: " .. #buff_list)

-- 给角色添加BUFF
lualib:AddBuff(player, "加速BUFF")

-- 删除角色BUFF
lualib:DelBuff(player, "加速BUFF")
```

#### 角色移动和跳转
```lua
-- 让玩家跳转到指定地图坐标
lualib:Player_MapMoveXY(player, "龙城", 100, 100)

-- 让玩家回城
lualib:Player_GoHome(player)

-- 让玩家跳转到目标玩家面前
local target_guid = lualib:Name2Guid("目标角色名")
lualib:Player_JumpToRole(player, target_guid)
```

#### 角色社交操作
```lua
-- 获取玩家行会信息
local guild_name = lualib:GetFamilyName(player)
local guild_guid = lualib:GuildGuid(player)
lualib:Warn("行会名称: " .. guild_name .. ", 行会GUID: " .. guild_guid)

-- 判断玩家是否有队伍
if lualib:Player_HasTeam(player) then
    lualib:Warn("玩家有队伍")
else
    lualib:Warn("玩家没有队伍")
end
```

## 物品接口使用示例

### 物品管理

#### 获取和设置玩家物品
```lua
-- 获取玩家所有物品列表
local items = lualib:AllItemList(player)
lualib:Warn("玩家物品数量: " .. #items)

-- 给玩家添加物品
lualib:AddItem(player, "大角羊", 300, false, "添加物品", "系统")

-- 给玩家添加单个物品
local item_guid = lualib:AddSingleItem(player, "大角羊", 1, 0, "添加单个物品", "系统")
lualib:Warn("新添加物品GUID: " .. item_guid)

-- 获取玩家物品数量
local count = lualib:ItemCount(player, "大角羊")
lualib:Warn("玩家大角羊数量: " .. count)

-- 删除玩家物品
lualib:DelItem(player, "大角羊", 100, 2, "删除物品", "系统")
```

#### 使用物品
```lua
-- 让玩家使用道具
local use_success = lualib:ApplyItem(player, "小飞鞋", 1, false, "使用道具", "系统")
if use_success then
    lualib:SysMsg_SendWarnMsg(player, "道具使用成功")
else
    lualib:SysMsg_SendWarnMsg(player, "道具使用失败")
end

-- 让玩家对目标使用道具
local target_guid = lualib:Name2Guid("目标角色名")
local apply_success = lualib:ApplyItemTarget(player, "治疗药水", false, target_guid)
if apply_success then
    lualib:SysMsg_SendWarnMsg(player, "道具使用成功")
else
    lualib:SysMsg_SendWarnMsg(player, "道具使用失败")
end
```

#### 获取物品属性
```lua
-- 获取物品信息
local item_name = lualib:Item_ChangeName(item_guid, "新物品名称")
local item_count = lualib:Item_GetAmount(item_guid)
local item_level = lualib:Item_GetLevel(item_guid)
lualib:Warn("物品名称: " .. item_name .. ", 数量: " .. item_count .. ", 等级: " .. item_level)

-- 判断物品是否绑定
local is_bind = lualib:Item_IsBind(item_guid)
if is_bind then
    lualib:Warn("物品是绑定的")
else
    lualib:Warn("物品是未绑定的")
end
```

#### 物品操作
```lua
-- 丢弃物品
lualib:DropItem(player, item_guid)

-- 移动物品
lualib:MoveItem(player, item_guid, target_site)

-- 判断物品是否在包裹中
local in_bag = lualib:InBag(player, item_guid)
if in_bag then
    lualib:Warn("物品在包裹中")
else
    lualib:Warn("物品不在包裹中")
end
```

## 怪物接口使用示例

### 怪物管理

#### 生成和移除怪物
```lua
-- 在地图上生成怪物
lualib:Map_GenMonster(map_guid, "僵尸", 100, 100)

-- 批量生成怪物
lualib:Map_BatchGenMonster(map_guid, "僵尸", 5, 100, 100, 200, 200)

-- 清理地图上的怪物
lualib:Map_ClearMonster(map_guid, 100, 100, 200, 200)

-- 移除指定怪物
lualib:Monster_Remove(monster_guid)
```

#### 怪物属性操作
```lua
-- 获取怪物信息
local monster_name = lualib:Name(monster_guid)
local monster_type = lualib:Monster_Type(monster_guid)
local is_dead = lualib:Monster_IsDie(monster_guid)
lualib:Warn("怪物名称: " .. monster_name .. ", 类型: " .. monster_type .. ", 是否死亡: " .. tostring(is_dead))

-- 设置怪物重生时间
lualib:Monster_SetRebornTime(monster_guid, 30000) -- 30秒后重生

-- 获取怪物重生时间
local reborn_time = lualib:Monster_GetRebornTime(monster_guid)
lualib:Warn("怪物重生时间: " .. reborn_time .. "毫秒")
```

#### 怪物行为控制
```lua
-- 让怪物攻击目标
lualib:Monster_SetAttackTarget(monster_guid, target_guid)

-- 让怪物施放技能
lualib:Monster_SpellTarget(monster_guid, "火球术", target_guid)

-- 让怪物移动到指定位置
lualib:Monster_Escape(monster_guid, 150, 150)

-- 获取怪物攻击目标
local attack_target = lualib:Monster_GetAttackTarget(monster_guid)
if attack_target ~= "" then
    lualib:Warn("怪物攻击目标: " .. attack_target)
end
```

#### 怪物仇恨管理
```lua
-- 获取怪物仇恨列表
local hate_list = lualib:Monster_GetHateList(monster_guid)
lualib:Warn("怪物仇恨列表大小: " .. #hate_list)

-- 设置怪物仇恨列表
local new_hate_list = { {target_guid1, 100}, {target_guid2, 50} }
lualib:Monster_SetHateList(monster_guid, new_hate_list)

-- 增加怪物对目标的仇恨值
lualib:Monster_AddTargetHate(monster_guid, target_guid, 10)

-- 减少怪物对目标的仇恨值
lualib:Monster_SubTargetHate(monster_guid, target_guid, 5)

-- 清除怪物仇恨列表
lualib:Monster_ClearHateList(monster_guid)
```

## 副本地图接口使用示例

### 副本管理

#### 创建和销毁副本
```lua
-- 创建副本地图
local dgn_guid = lualib:Map_CreateDgn("副本地图", 600000) -- 副本持续10分钟
lualib:Warn("副本创建成功，GUID: " .. dgn_guid)

-- 判断地图是否为副本
local is_dgn = lualib:Map_IsDgn(dgn_guid)
if is_dgn then
    lualib:Warn("地图是副本")
else
    lualib:Warn("地图不是副本")
end

-- 设置副本剩余时间
lualib:Map_SetDgnLeftTime(dgn_guid, 300000) -- 设置为5分钟

-- 获取副本剩余时间
local left_time = lualib:Map_GetDgnLeftTime(dgn_guid)
lualib:Warn("副本剩余时间: " .. left_time .. "毫秒")

-- 销毁副本地图
lualib:Map_DestroyDgn(dgn_guid)
```

#### 副本进入和管理
```lua
-- 给玩家设置副本门票
lualib:Player_SetDgnTicket(player, dgn_guid)

-- 判断玩家是否拥有副本门票
local has_ticket = lualib:Player_HasDgnTicket(player, dgn_guid)
if has_ticket then
    lualib:Warn("玩家拥有副本门票")
else
    lualib:Warn("玩家没有副本门票")
end

-- 让玩家进入副本
lualib:Player_EnterDgn(player, dgn_guid)

-- 获取玩家所在的副本地图
local player_dgn = lualib:Player_GetDgnByTicket(player)
if player_dgn ~= "" then
    lualib:Warn("玩家在副本: " .. player_dgn)
end

-- 清理副本中的实体
lualib:Map_ClearDgnItem(dgn_guid)
lualib:Map_ClearDgnMonster(dgn_guid)
lualib:Map_ClearDgnNpc(dgn_guid)
lualib:Map_ClearDgnPlayer(dgn_guid)
```

## 行会接口使用示例

### 行会管理

#### 行会创建和管理
```lua
-- 创建行会
local family_guid = lualib:CreateFamily(player, "天下第一帮", "行会公告")
if family_guid ~= "" then
    lualib:Warn("行会创建成功")
else
    lualib:Warn("行会创建失败")
end

-- 删除行会成员
lualib:DelFamilyMember(family_guid, target_player)

-- 解散行会
lualib:DestroyFamily(family_guid)

-- 修改行会名称
lualib:Family_ChangeName(family_guid, "新行会名称")

-- 修改行会公告
lualib:ChangeFamilyNotice(family_guid, "新的行会公告")
```

#### 行会信息获取
```lua
-- 获取服务器上所有行会列表
local all_families = lualib:GetAllFamilys()
lualib:Warn("服务器行会数量: " .. #all_families)

-- 获取行会信息
local family_info = lualib:GetFamilyInfo(family_name)
lualib:Warn("行会名称: " .. family_info["Name"])
lualib:Warn("行会会长: " .. family_info["Leader"])
lualib:Warn("行会成员数量: " .. family_info["MemberCount"])

-- 获取行会成员列表
local members = lualib:GetFamilyMembers(family_guid)
lualib:Warn("行会成员数量: " .. #members)
for i, member_guid in ipairs(members) do
    lualib:Warn("成员" .. i .. ": " .. member_guid)
end

-- 获取玩家所在行会
local player_family = lualib:GetFamilyName(player)
if player_family ~= "" then
    lualib:Warn("玩家所在行会: " .. player_family)
else
    lualib:Warn("玩家未加入行会")
end
```

#### 行会战争
```lua
-- 开始行会战争
lualib:StartFamilyWar(family_guid1, family_guid2)

-- 判断两个行会是否处于战争状态
local has_war = lualib:HasFamilyWar(family_guid1, family_guid2)
if has_war then
    lualib:Warn("两个行会正在交战")
else
    lualib:Warn("两个行会未交战")
end

-- 获取正在进行的行会战争列表
local running_wars = lualib:GetRunningFamilyWar()
lualib:Warn("正在进行的行会战争数量: " .. #running_wars)

-- 停止行会战争
lualib:StopFamilyWar(family_guid1, family_guid2)
```

## 社交接口使用示例

### 聊天和组队

#### 聊天功能
```lua
-- 以指定角色身份进行喊话
lualib:Player_Chat(player, "世界", "大家好，我是新手")

-- 设置玩家禁言
lualib:Player_SetForbidTalk(player, "刷屏", 300) -- 禁言5分钟
```
